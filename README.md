# KiranaKart 🥦

A modern, local e-commerce application for connecting customers with nearby Kirana stores.

## Features (Phase 1 MVP)

- **Premium UI**: Fresh, vibrant design system using CSS variables and Glassmorphism.
- **Home Page**: Hero banner, Category browsing, and Featured Shops.
- **Login**: Mobile-first OTP login interface.
- **Routing**: Setup with React Router v6.

## Tech Stack

- **Frontend**: React + Vite
- **Styling**: Vanilla CSS (Premium Custom Design System)
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`

## Project Structure

- `src/components`: Reusable UI components (Navbar, etc.)
- `src/pages`: Page views (Home, Login)
- `src/layouts`: Layout wrappers
- `src/assets`: Images and static assets
- `src/index.css`: Global design system and theme variables
